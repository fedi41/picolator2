PICOLATOR is a handheld device powered by a Raspberry Pi Pico, featuring a custom operating system.

Hardware

Picolator runs on a Raspberry Pi Pico (2W), chosen for its small size and full low-level control.

It drives a 1.3" LCD display (Waveshare module) that is directly wired to the board, keeping the system simple and fast.

All interaction is handled through physical GPIO buttons, making the device feel fully handheld and self-contained.

Firmware

The system is written in C++ using the Raspberry Pi Pico SDK, giving full control over the hardware without any heavy abstraction layers.

On top of that, Picolator runs a lightweight custom system that handles input, rendering, and UI logic directly.

This approach keeps everything fast, predictable, and easy to extend as new features are added.

Features

The UI system includes blend modes, allowing visual effects and more flexible rendering even on limited hardware.

Alongside that, a built-in Settings app lets you toggle features and switch themes, keeping the system configurable without touching code.

For interaction and testing, Picolator also includes a simple Dino Runner game, which demonstrates input handling and real-time updates.

Overall, the system is designed to grow over time, with space for new apps and internal features.

Design

Picolator is built around a simple idea: exploring how much you can do with very little hardware.

Instead of relying on frameworks, everything is built directly on top of the system for full control and understanding.
