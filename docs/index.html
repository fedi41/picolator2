PICOLATOR is a handheld device powered by a Raspberry Pi Pico, featuring a custom operating system.
> GitHub < > Devlogs < > Releases < > Gallery <

--- This project was originally built for Stardance ---

Hardware

  • Powered by a Raspberry Pi Pico 2 W.
  • 1.3" Waveshare LCD display.
  • Physical GPIO buttons for input.

Firmware

  • Written in C++.
  • Built with the Raspberry Pi Pico SDK.
  • Custom UI and rendering engine.
  • Lightweight and modular design.

Features

  • Custom UI with blend modes.
  • Settings app with feature toggles.
  • Theme selection.
  • Built-in Dino Runner game.
  • More features coming soon.

Design

  • Built from scratch.
  • Focused on simplicity.
  • Full hardware control.
  • Open for experimentation.
